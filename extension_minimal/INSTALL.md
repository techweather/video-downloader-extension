# dlwithit Firefox Extension — Installation

## Quick Install

1. Double-click `dlwithit-extension-1.0.0.xpi`
2. Firefox will prompt you — click **Add**
3. The dlwithit icon will appear in your toolbar

## Alternative: Load via Firefox

1. In Firefox, go to **File → Open File** and select the `.xpi` file
2. Click **Add** when prompted

> **Note:** dlwithit.app must be installed and running for the extension to work.
> Look for the dlwithit icon in your menu bar.
